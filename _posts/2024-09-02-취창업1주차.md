# Basics 
Time Complexity <br/>
O(log(N)) < O(N) < O(Nlog(N)) < O(N2) <br/>
(상수 영향 무시, N은 매우 큰 값) <br/>
Example1: <br/>
```
#include <iostream>
using namespace std;
int main()
{
  int N=100
  int cnt=0;
  for(int i=0; i < N; i++)
  {
    for(int j=0; j < N;j++)
    {
      cout << "연산횟수:" << cnt++ << "\n"
    }
  }
}
```
=> O(N^2) <br/>

int = 4byte  = 325 <br/>
long = 8byte <br/>
